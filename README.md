### Решение:

##### Docker Compose - инструмент для развертывания и управления многоконтейнерными приложениями. Он позволяет запускать, останавливать и обновлять сервисы приложения используя файл с описанием инструкций этого приложения, такие как: образы, сети, тома. Вместо ручного запуска каждого контейнера по отдельности Docker Compose позволяет запустить все контейнеры в правильной последовательности одной командой, имея корректный dockerfile. В своей работе я использую Docker Compose для оперативного обновления демонстрационного стенда разрабатываемого нашей организации ПО, достаточно загрузить образ со свежей версией ПО в Docker, запустить docker-compose up -d, и свежая версия ПО уже доступна для использования, весь процесс занимает не более 5 минут.
